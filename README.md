
# Dockerized Web Server (Flask)

## Overview
A simple Python Flask web application containerized using Docker and exposed on port 5000.

## How to Run

1. Build the image:
   docker build -t flask-webserver .

2. Run the container:
   docker run -d -p 5000:5000 flask-webserver

3. Open your browser:
   http://localhost:5000
